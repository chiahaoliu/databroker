Data Broker¶

The databroker Python package provdies a simple, user-friendly interface for retireving stored data and metadata. It retrieves data into memory as built-in Python data types and numpy arrays.

Both simple and complex searches are supported. Convenience fucntions provide a simple way to handle tabular data and image sequences.

Examples of Basic Usage¶

If they are not already imported, import DataBroker and its convenience fucntions. (This step may already be done as part of an IPython profile.)

from databroker import DataBroker as db, get_table, get_images, get_events

Example 1: Load data in a table.¶

last_run = db[-1]
table = get_table(last_run)

To exclude non-scalar data* like images, which take time to load and are usually useful to view in tabular form, use the fill keyword like so:

table = get_table(last_run, fill=False)

*To be precise, fill=False excludes externally stored data, which almost always means larges files and array data.

Example 2: Load select fields of data.¶

For faster loading, restrict the table to include only certain columns.

last_run = db[-1]
table = get_table(last_run, ['temperature', 'motor_position'])

Example 3: Load images.¶

last_run = db[-1]
images = get_images(last_run, 'ccd_image')

To plot the first image using matplotlib:

import matplotlib.pyplot as plt
img = images[0]
plt.imshow(img)

See Searching for Data and Fetching Data for details.
